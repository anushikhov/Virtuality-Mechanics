
1. **Programming, C++, and the build toolchain** — what programs are, what C/C++ are, compiler/linker/libraries, IDE setup, first build, warnings, standards, and build configs.   

2. **Program structure and statements** — statements, comments, formatting, and the basic shape of a C++ program.   

3. **Variables, objects, initialization, literals, expressions** — variables, assignment, initialization, operators, expressions, identifier rules, undefined behavior.   

4. **Console input/output basics** — `cout`, `cin`, `endl`, and basic interaction with the user.   

5. **Functions and parameter passing basics** — return values, void functions, parameters, arguments, local scope, and why functions matter.   

6. **Multi-file programs, headers, preprocessor, and namespaces** — forward declarations, multiple files, naming collisions, headers, header guards, and the preprocessor.   

7. **Debugging and development workflow** — syntax vs semantic errors, debugging strategy, debugger use, breakpoints, stepping, watch windows, call stack, preventing bugs early.   

8. **Fundamental data types** — `void`, integer types, floating point, booleans, chars, object sizes, `sizeof`, fixed-width integers, `size_t`.   

9. **Constants, literals, numeral systems, and compile-time evaluation** — named constants, literal forms, binary/hex/octal, constant expressions, `constexpr` variables, as-if rule, compile-time optimization.   

10. **Strings and views of strings** — `std::string`, `std::string_view`, and how text handling differs from raw character arrays.   

11. **Operators and bit manipulation** — precedence, arithmetic, remainder, increment/decrement, comma, conditional, relational/logical operators, bitsets, bitwise ops, masks, binary/decimal conversion.   

12. **Scope, duration, linkage, globals, and namespaces** — blocks, local/global variables, shadowing, internal/external linkage, static locals, inline variables, using declarations, unnamed/inline namespaces.   

13. **Control flow** — `if`, `constexpr if`, `switch`, fallthrough, `goto`, loops, `break`, `continue`, halting early.   

14. **Random number generation** — intro RNG, Mersenne Twister, and global RNG helpers.   

15. **Testing, assertions, and input/error handling** — testing, code coverage, semantic mistakes, detecting errors, bad input handling, `assert`, `static_assert`.   

16. **Type conversion, aliases, and type deduction** — implicit/explicit conversion, promotions, narrowing, arithmetic conversions, `static_cast`, `typedef`, aliases, `auto`, function type deduction.   

17. **Function overloading and function templates** — overload sets, overload resolution, ambiguous calls, deleted functions, default args, function templates, instantiation, multiple template parameters, non-type template parameters.   

18. **`constexpr` / `consteval` functions** — compile-time-capable functions and the rules around constant evaluation.   

19. **References, pointers, value categories, and optional values** — lvalues/rvalues, lvalue refs, const refs, pass-by-ref, pointers, null pointers, const with pointers, pass-by-address, returning refs/addresses, `std::optional`.   

20. **User-defined types: enums, structs, and basic class templates** — unscoped/scoped enums, enum conversions, enums to strings, structs, aggregate initialization, member defaults, passing/returning structs, CTAD, alias templates.   

21. **Classes and OOP fundamentals** — OOP intro, classes, member functions, const objects/member functions, access control, accessors, encapsulation, constructors, initializer lists, copy constructor, copy elision, `explicit`, `constexpr` classes.   

22. **More class mechanics** — `this` pointer, chaining, classes in headers, nested types, destructors, static members, friend functions/classes, ref qualifiers.   

23. **Containers and arrays** — containers overview, `std::vector`, passing/returning vectors, move intro through vectors, resizing/capacity, range-for, `std::array`, C-style arrays, array decay, pointer arithmetic, C-style strings, multidimensional arrays.   

24. **Iterators, algorithms, sorting, and timing** — selection sort, iterators, standard algorithms, measuring performance.   

25. **Dynamic memory and low-level allocation** — `new`/`delete`, dynamic arrays, heap use, pointers-to-pointers, dynamic multidimensional arrays, void pointers.   

26. **Advanced function topics** — function pointers, stack vs heap, recursion, command-line arguments, ellipsis, lambdas, captures.   

27. **Operator overloading and copy behavior** — arithmetic/comparison/I/O/subscript/call/typecast/assignment overloads, unary ops, shallow vs deep copy, overloads with templates.   

28. **Move semantics, smart pointers, and object relationships** — rvalue references, move ctor/assignment, `std::move`, `unique_ptr`, `shared_ptr`, `weak_ptr`, composition, aggregation, association, dependency, container classes, `initializer_list`.   

29. **Inheritance and runtime polymorphism** — inheritance basics, construction order, overriding, access in inheritance, hiding functionality, multiple inheritance, base pointers/references, virtual functions, `override`/`final`, virtual destructors, vtables, abstract classes, virtual bases, slicing, `dynamic_cast`.   

30. **Advanced templates, exceptions, I/O systems, libraries, and language evolution** — class template specialization, exceptions and `noexcept`, stream I/O, file I/O, static vs dynamic libraries, using libraries in IDEs, language reference use, and C++11/14/17/20/23 update overviews.
